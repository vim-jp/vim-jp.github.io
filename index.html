vim-jp は Vim と日本・日本語に関わるあらゆるリソースを集中させることを目的としたコミュニティサイトです。

Vim の元となったvi は、おそらく全てのUNIX系 OS 上で使えるテキストエディタです。世界には、さまざまなテキストエディタが存在し、有名なところでは Emacs/XEmacs、 Windows のメモ帳(notepad)、Mac の TextEdit などがあります。大量の文章をタイプしない方であれば、メモ帳(notepad)で十分かもしれません。しかしプロのソフトウェア開発者や敏速な設定変更を余儀なくされる管理者の方にとって、使い慣れたテキストエディタが無いということは命取りとも成りかねません。魅力的なエディタはいくつもありますが、その中でも少ないキーストロークでの編集作業や、カスタマイズ性に富んだエディタ が vi です。vi は Java、BSD、csh 等で有名な Bill Joy 氏が 1976 年に 作り始め、その後は UNIX 系 OS では標準のエディタとして君臨して来ました。

その後、 vi クローンと呼ばれる vi に似せたエディタが登場し始めました。 その中でも Bram Moorenaar 氏によって1980年代後半から開発された Vim (当初名前はVi IMitationでした) は vi が持つ機能の 99% を持ち合わせ、かつ vi を上回る編集能力 を備えています。

Vim は今や開発者やサーバ管理者には無くてはならない存在であり、私達 vim-jp は Vim の普及と開発の存続をサポートする為にユーザと Vim 開発者の渡し舟を担うべく誕生しました。日本の Vimmer がより活発に活動出来るよう、皆様からの不具合報告や情報提供をお待ちしています。
